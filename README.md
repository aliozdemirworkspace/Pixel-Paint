# Pixel Paint

Üstteki kirli dokuyu sprey ile temizleyip alt katmandaki piksel sanatını ortaya çıkaran tek dosyalık tarayıcı oyunu.

## Yerel önizleme

Oyun dosyası `public/index.html` altındadır. Yerel önizleme:

```bash
cd public && python3 -m http.server 8080
```

Sonra `http://localhost:8080` adresine gidin.

## Vercel

Bu repo `index.html` kökünde statik dosya olarak deploy edilecek şekilde hazırlanmıştır. Vercel’de projeyi import edip deploy etmeniz yeterlidir.
